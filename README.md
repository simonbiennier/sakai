## Getting Started

Clone the repo

```sh
git clone https://github.com/simonbiennier/sakai.git
```

Install packages

```sh
yarn
```

Then, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
